# Drowsiness Detection System

Real-time driver drowsiness and distraction detection using:
- OpenCV - video capture and frame processing
- MediaPipe - face and eye landmark detection
- YOLOv8n - trained drowsiness classifier
- Ollama LLaMA 3.2 Vision - edge case analysis
- FastAPI - local offline interface

## Run Locally
Works 100% offline. No internet needed after setup.
